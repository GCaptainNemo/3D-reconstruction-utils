# 三维重建工具箱

## 一、介绍

依赖库: `PCL` +` Eigen`

本仓库保存一些平时写的关于三维重建的工具

## 二、功能

* 利用`PCL`的可视化模块，绘制相机轨迹。(src/draw_trajectory.cpp)
* 深度图转点云 (src/range2pc.cpp)
